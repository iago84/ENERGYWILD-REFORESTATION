El sistema NEXUS RISING proporciona el marco metodológico validado para este

| Componente NEXUS | Aplicación a Carbono Negativo |

| Nodos territoriales | Parcelas de captura de CO₂ |
| Sensores IoT | Monitoreo de biomasa, suelo, agua |
| Microredes energéticas | Energía renovable para operaciones |
| ML/AI optimization | Optimización de plantaciones y captura |
| Blockchain tokens | Certificados de carbono tokenizados |
