## Prompt 1: Análisis de Especies Xerofíticas Productivas
```
Analiza especies xerofíticas productivas para zonas áridas del Sahel:
- Opuntia ficus-indica (nopal): eficiencia hídrica, rendimiento comestible
- Cucurbitáceas (sandía, melón): producción hídrica biológica
- Moringa oleifera: densidad nutricional extrema
- Acacia spp.: fijación de nitrógeno, infraestructura hídrica
- Neem: control biológico de plagas

Para cada especie, proporcione:
1. Requerimientos hídricos (L/kg producción)
2. Tiempo de establecimiento
3. Beneficios para el suelo
4. Valor comercial estimado
```


## Prompt 3: Optimización de Captación de Agua
```
Optimiza captación de agua atmosférica en zona ventosa árida:
- Comparativa: fog harvesting vs condensación radiativa vs higroscópicos
- Diseño de malla óptima (material, orientación, altura)
- Rendimiento esperado (L/m²/día)
- Integración con sistema de almacenamiento subterráneo

Considera:
1. Dirección del viento dominante (NE-SW Harmattan)
2. Humedad relativa mínima operativa
3. Materiales locales disponibles
```

## Prompt 4: Modelo de Expansión Territorial 100 Años
```
Modela expansión territorial de 10-100 años para sistema de nodos:
- Dinámica logística: dN/dt = rN(1-N/K)
- Balance hídrico: ΔW = P + C + G - E - T - U
- Crecimiento vegetal: dV/dt = αW - βErosión

Variables:
- N(t): número de nodos
- V(t): vegetación estable
- W(t): agua disponible
- S(t): estabilidad del suelo

Salida: mapa de expansión probabilístico con puntos críticos.
```

## Prompt 6: Geotextil Reciclado vs Biodegradable
```
Comparativa técnica: geotextil reciclado vs biodegradable:
- Fuerza de tracción (kN/m)
- Vida útil en campo (años)
- Coste de producción (€/m²)
- Impacto microplástico
- Ciclo de degradación controlada

Recomendación: uso híbrido fase 0-5 años (plástico) → fase 5-15 años (vegetación).
```
