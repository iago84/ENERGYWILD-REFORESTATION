# ENERGYWILD

Sistemas integrados de regeneración territorial y generación energética para zonas áridas.

## Componentes principales

- PIRS-1 — Programa Integrado de Regeneración del Sahel (agua, suelo, agricultura, microredes)
- BSLC Magnetic Generator — Generación eléctrica por conmutación magnética con IA
- NEXUS RISING Node Network — Arquitectura de red de nodos con ML/optimización

## Estructura

```
01-energia/                 # BSLC + microredes energéticas + documentación técnica
     generador-bslc/        # Sub-repo generador BSLC (diagramas, prompts, modelos)
pirs1-sistema/              # Código PIRS-1 (modelos, simulación, API)
02-reforestacion/           # Regeneración forestal (verificación futura)
03-agua/                    # Captación y gestión hídrica
04-agricultura/             # Sistemas agrícolas regenerativos
05-industria-mallas/        # Geotextiles y mallas de retención
CARBONO-NEGATIVO-HACIENDA-SAN-JUAN.md  # Aplicación NEXUS RISING a proyectos de carbono negativo
```
