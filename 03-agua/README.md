# Sistema Hídrico - ESTRATEGIA BSLC

## Visión General
Captura y multiplicación del agua atmosférica + gestión integral del agua subterránea.

## Arquitectura Hídrica Híbrida

### A. Captación Atmosférica Activa
1. **Deshumidificadores solares de baja energía** (adsorción con zeolitas o sales higroscópicas)
2. **Torres de condensación nocturna** (radiative cooling)
3. **Redes de "fog harvesting"** donde sea viable

### B. Captación Pasiva
1. **Superficies hidrofílicas inclinadas**
2. **Materiales cerámicos de condensación térmica**
3. **Techos de doble fase** (captura + almacenamiento)

### C. Mejor Sistema para Zonas Ventosas
**Fog Harvesting** - Mallas verticales capturan microgotas
- Eficiencia: 2-20 L/m²/día
- Ventaja: El viento es motor, no enemigo
- Ideal para zonas ventosas del Sahel

## Exploración de Acuíferos

### Técnicas
- Prospección geofísica (resistividad eléctrica)
- Radar de penetración de suelo
- Mapas de paleocanales fósiles
- Micro-perforación modular (pozos distribuidos en red)

## Almacenamiento y Estabilización

### Opciones
1. **Cisternas enterradas** - baja evaporación
2. **Acuíferos artificiales recargados** (MAR systems)
3. **Biofiltración** - capas de grava, arcilla y carbón vegetal

## Balance Hídrico del Sistema
```
ΔW = P + C + G − E − T − U

Donde:
P = lluvia
C = condensación atmosférica
G = aporte subterráneo
E = evaporación
T = transpiración vegetal
U = uso humano/agro
```

**Objetivo**: ΔW ≥ 0 (sistema hídrico estable)

## Terraformación Hídrica Suave

### Técnicas
- Incremento de humedad local mediante vegetación estratégica
- Corredores verdes de evapotranspiración
- Sombra estructural (árboles + estructuras ligeras)

### No se altera el planeta
Se re-escribe el microclima para favorecer la retención de humedad.
