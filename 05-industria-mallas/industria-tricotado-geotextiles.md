# INDUSTRIA LOCAL: TRICOTADO Y GEOTEXTILES RECICLADOS
## Modelo de fábrica comunitaria de mallas para el Sahel

---

## 1. CONTEXTO Y OPORTUNIDAD
- Demanda local: mallas para fog harvesting + protección erosión + cercos vivos.
- Oportunidad de inserción laboral: mujeres y jóvenes.
- Modelo: taller semimecanizado, mantenible por comunidad.

---

## 2. MATERIA PRIMA

### 2.1 Origen
- **PET reciclado** (botellas, envases).
- Proveedor: plantas de reciclado urbano + acopio comunitario.
- Calidad requerida: limpio, seco, separado por color (verde/transparente preferible).

### 2.2 Procesado
| Paso | Descripción | Equipo |
|------|-------------|--------|
| Triturado | A 5-10 mm | Molino manual / eléctrico |
| Lavado | Agua + detergente biodegradable | Tina + filtro |
| Secado | Solar o forzado 40-60°C | Invernadero / estufa solar |
| Hilado | Extrusión a filamentos | Extrusora manual / semiautomática |
| Bobinado | Carrete soporte | Bastidor simple |

### 2.3 Coste materia prima (orientativo)
- PET recogido: 20-50 €/tn (casi gratuito en comunidad organizada).
- Energía proceso: 0,5-1,5 kWh/kg.
- Químicos: 0,1 €/kg producto.

---

## 3. MÁQUINA DE TRICOTADO

### 3.1 Tipo recomendado
- **Máquina de tejer plana (warp knitting)** simplificada.
- Modelo community-grade: 1-2 operarios, baja velocidad.
- Capacidad: 50-200 m²/día según modelo.

### 3.2 Funciones
- producir mallas Raschel 3D para fog harvesting.
- Tejer refuerzos para geotextil checkerboard.
- Fabricar tejidos para protección solar/cultivo.

### 3.3 Proveedores referencia
- India / China: máquinas pequeñas 10.000-30.000 €.
- Opción DIY: telares manuales de bajo coste (fase inicial).

---

## 5. MODELO ECONÓMICO TALLER

### 5.1 Inversión inicial (fase 1)
| Item | Coste € |
|------|---------|
| Triturador + lavado | 3.000-8.000 |
| Extrusora básica | 5.000-15.000 |
| Telar manual/warp-knitting | 2.000-10.000 |
| Herramientas + moldes | 1.000-2.000 |
| **Total** | **11.000-35.000 €** |

### 5.2 Coste variable
- Energía: 0,3 €/m²
- Mantenimiento: 5-10 % año
- Mano obra: 2-4 operarios × 8h/día

### 5.3 Ingresos anuales (escala año 2-3)
- 1000 m² vendidos × 3-8 €/m² = 3.000-8.000 €/año
- Payback: 2-5 años dependiendo demanda.

---

## 6. CADENA DE VALOR LOCAL

```
Recogida PET → Triturado → Lavado → Secado → Hilado → Tejido → Distribución
     ↑              ↑         ↑         ↑         ↑         ↑         ↑
Comunidad    Taller local   Solar   Taller    Taller    Taller   Nodos BSLC + mercado
```

### Beneficios
- Cierre local de residuos plásticos.
- Empleo cualificado (mujeres + jóvenes).
- Reducción coste mallas (70% vs importación).
- Producto alineado con ODS 8, 9, 12, 13, 15.

---

## 7. PLAN DE ESCALADO
- **Año 1**: 1 taller piloto (1000 m²/año).
- **Año 2-3**: 3 talleres regionales (3000 m²/año total).
- **Año 4-7**: Cooperativa de productores de geotextiles.
- **Año 8+**: Exportación regional (Sahel).
