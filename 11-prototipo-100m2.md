## Cronograma Construcción (14 días)

### Día 1-2: Preparación Sitio
```
TAREAS:
☐ Localizar sitio 10×10m (GPS)
☐ Limpiar vegetación existente
☐ Marcar puntos esquina con estacas
☐ Excavar cisterna 6×6m (60cm profundidad)
☐ Verificar drenaje natural
☐ Tomar fotos antes

MATERIALES USADOS:
- Estacas 4 unidades
- Palas 2 unidades
- Metro 2 unidades
```

### Día 3-4: Cisterna + Bomba
```
TAREAS:
☐ Colocar geomembrana base
☐ Sellar juntas con termo (150°C)
☐ Montar estructura hormigón armado
☐ Instalar entrada/agua residual 10cm
☐ Conectar bomba solar sumergible
☐ Prueba llenado/ vaciado

MATERIALES USADOS:
- Geomembrana 36 m²
- Tubo PVC 4m
- Bomba solar 1 unidad
```

### Día 5-6: Torre Vertical
```
TAREAS:
☐ Cortar perfiles acero 3m altura
☐ Soldar 3 plataformas (0.8m, 1.8m, 2.8m)
☐ Montar estructura 10×10cm base
☐ Instalar sistema riego microaspersión
☐ Probar bomba por gravedad

MATERIALES USADOS:
- Acero galvanizado 5 kg
- Tubo riego 50m
- Gotero micro 20 uds
```

### Día 7-8: Sistema Solar
```
TAREAS:
☐ Montar soporte inclinado 15° norte
☐ Colocar paneles 300W (2 uds)
☐ Cableado DC 6mm²
☐ Instalar controlador MPPT
☐ Conectar batería LFP
☐ Prueba carga/descarga

MATERIALES USADOS:
- Panels solares 2 uds
- Batería LFP 1 ud
- Controlador MPPT 1 ud
- Cable 10m
```

### Día 9-10: Mallas Anti-Erosión
```
TAREAS:
☐ Desplegar mallas 2×2m (10 uds)
☐ Fijar estacas madera cada 2m
☐ Conectar mallas con clips
☐ Enterrar borde 10cm tierra
☐ Verificar cobertura checkerboard

MATERIALES USADOS:
- Mallas geotextil 40 m²
- Estacas madera 20 uds
- Clips plástico 40 uds
```

### Día 11-12: Suelo Regenerado
```
TAREAS:
☐ Excavar 80cm profundidad
☐ Colocar capa arcilla compactada (10cm)
☐ Aplicar biochar (40cm)
☐ Añadir compost (20cm)
☐ Inocular micorrizas
☐ Registrar pH inicial

MATERIALES USADOS:
- Biochar 2 m³
- Compost 2 m³
- Arcilla local 1 m³
- Micorrizas 1L
```

### Día 13-14: Cultivo + Pruebas
```
TAREAS:
☐ Sembrar nopal en perímetro
☐ Plantar mijo zona abierta
☐ Instalar sensor humedad
☐ Configurar Arduino + SD
☐ Primer riego bomba solar
☐ Registro fotos finales

MATERIALES USADOS:
- Semillas nopal 1kg
- Semillas mijo 0.5kg
- Sensor humedad 1 ud
- Arduino kit 1 ud
```

## Pruebas Funcionales Piloto

### Agua (Día 14)
- [ ] Cisterna llena sin filtraciones (<1%)
- [ ] Bomba funciona 8h/día con panel
- [ ] Mallas 100% cobertura efectiva
- [ ] Producción agua: >10 L/día

### Energía (Día 14)
- [ ] Paneles 300W cargan batería a 80%
- [ ] Batería 12V 100Ah 5 días autonomía
- [ ] LED 5W ilumina 4 horas nocturnas
- [ ] Sensor Arduino registra datos

### Cultivo (Día 14)
- [ ] 50 nopal plantados, 0% mortalidad
- [ ] 10 m² mijo sembrado
- [ ] pH suelo 7-8
- [ ] Humedad 20-30% profundidad 30cm
